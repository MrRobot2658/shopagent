## 第一阶段：基建搭建

### Step 1 - 注册 Shopify 店铺

1. 打开视频简介中的 [Shopify 注册链接](https://shopify.pxf.io/BnBxny)
2. 输入邮箱 → 点「继续」
3. 设置密码 → 点「创建账号」
4. 选择所在地区（后期可改，随便选）
5. 进入 Shopify 后台
6. **订阅套餐**：3天免费试用，前3个月每月$1，之后$25/月
   - 新手选第一个套餐即可
   - 支持信用卡或 PayPal 付款
   - 如果不立即订阅，可先点右上角「跳过」
7. ✅ 语言切换：右上角头像 → 语言 → 简体中文 → 保存

### Step 2 - 注册 Zendrop（一件代发平台）

1. 打开视频简介中的 Zendrop 注册链接
2. 点击右上角「免费试用」
3. 填写：名、姓、邮箱、密码
4. 勾选同意条款 → 创建账号
5. 订阅页面点「跳过」/「暂时不需要」
   - ⚠️ Zendrop 目前活动价 $197/年
6. 进入 Zendrop 后台

### Step 3 - 连接 Zendrop ↔ Shopify

1. Zendrop 后台左上角 → 「连接店铺」
2. 选择 Shopify → 「已经有的店铺」→ 连接
3. 点击「安装 Zendrop」
4. 如果提示订阅 Shopify，完成订阅后再回来
   - 设置付款方式（PayPal 或信用卡）
   - 选择套餐（新手选 Basic $1/月）
5. 选择店铺品类（电子产品 / 不确定都可以）
6. 点击「一键发布」或 Let's Go
7. ✅ **完成连接**

## 第二阶段：AI 选品

### Step 4 - 配置 Claude AI + Zendrop MCP

前提：已注册 [Claude AI 账号](https://claude.ai)

1. 进入 Claude 界面
2. 左上角 → 打开侧边栏
3. 最底部 → 「自定义」
4. 点击「连接器」→ 「+ 增加自定义连接器」
5. 命名（如 "Zendrop"）
6. 粘贴 Zendrop MCP 地址（见视频简介/评论区）
7. 点击「连接」
8. 授权权限：
   - 读取产品信息 ✓
   - 管理导入产品 ✓
   - 查看订单 ✓
   - 取消不想要的权限（如不需要给订单权限可取消）
9. 点击「授权」

> 📌 **MCP 工作原理**：Claude ↔ Zendrop（MCP）↔ Shopify
> Claude 通过 MCP 接口访问 Zendrop 数据 → Zendrop 同步到 Shopify

### Step 5 - 用 AI 做选品研究

**准备提示词**（见附录「提示词1」）

操作流程：

1. 复制「提示词1」全文
2. 修改其中的「细分赛道」为你的品类（如：电子产品、宠物用品、运动户外）
3. 粘贴到 Claude 聊天窗口
4. 点击发送
5. Claude 会通过 MCP 拉取 Zendrop 产品库数据
6. 等待生成**爆品研究报告**（包含 19+ 分析维度）

**报告内容示例**：

| 字段 | 说明 |
|------|------|
| 产品名称 | 家用 IPL 激光脱毛仪 |
| 建议零售价 | $XX.XX |
| Zendrop 成本价 | $XX.XX |
| 毛利 | >200% |
| 情绪购买动机 | 分析（如：隐私解放、自信提升） |
| 目标受众 | 18-45岁女性 |
| 每日预算建议 | $50-$100（测试期）→ $500-$3,000（放量期） |
| UGC 拍摄角度 | 3-5个具体分镜建议 |

## 第三阶段：产品上架

### Step 6 - 用 AI 一键上架产品

在 Claude 中直接发送指令：

```
请把第一个产品增加到我的 Zendrop 导入列表
```

Claude 会自动：

1. 查询你的店铺 ID
2. 将产品添加到 Zendrop 导入列表
3. ✅ 确认操作成功

### Step 7 - 从 Zendrop 发布到 Shopify

**方式一：全自动（Claude 中操作）**

```
请把该产品发布到我的 Shopify 店铺
```

**方式二：半自动（Zendrop 后台）**

1. Zendrop → 我的产品 → 导入列表
2. 找到产品 → 点「⋯」→ 发布到店铺
3. 可编辑描述/图片后发布
4. 回到 Shopify → 产品 → 刷新 → 确认已出现

### Step 8 - 移除店铺密码（重要）

新 Shopify 店铺默认有密码保护，需要移除：

1. Shopify 后台 → 左边「在线商店」
2. 先添加店铺地址（用于移除密码）：
   - 设置 → 店铺详情 → 添加地址
3. 回到「在线商店」→ 关掉「设密码保护」
4. 右下角保存 → 此时店铺对所有人开放

## 第四阶段：市场深度研究

### Step 9 - ICP 分析与竞品研究

**准备提示词**（见附录「提示词2」）

操作流程：

1. 回到 Zendrop 后台 → 产品详情 → 复制产品链接
2. 修改提示词中的「产品名称」和「产品链接」
3. 粘贴到 Claude 中发送

**AI 分析报告包含**：

- ✅ 竞品分析（Facebook 广告主、品牌竞品）
- ✅ 评论分析
- ✅ 社会舆论、广告心理分析
- ✅ 消费者行为分析
- ✅ 核心购买动机
- ✅ 核心购买顾虑
- ✅ 消费者身份认同分析
- ✅ **5个差异化营销角度**（每个带评分）
  - 角度1：被忽略的隐秘痛苦 ⭐ 推荐
  - 角度2：XX 主题
  - 角度3：XX 主题
- ✅ 目标顾客画像

### Step 10 - 生成营销文案

选一个营销角度，在 Claude 中发送：

```
我喜欢第一个营销角度「被忽略的隐秘痛苦」，
请根据这个营销角度帮我写产品文案，
让它更好地突出这个情绪点以及产品真正解决的问题。
```

Claude 会生成：

- 网站文案（直接可用）
- 广告文案
- 视频脚本
- 多套方案可选

## 第五阶段：广告素材制作

### 方式 A - Zendrop 内置 AI 广告生成

1. Zendrop 后台 → AI 广告生成
2. 将 Claude 生成的广告文案粘贴进去
3. 系统用数字人自动生成产品短视频
4. 需要升级 Zendrop 会员（主流套餐含 525 积分）

### 方式 B - Quickfire（推荐专业工具）

> Quickfire 是专业的电商广告视频生成平台

操作流程：

1. 打开 Quickfire（链接见视频简介）
2. 用邮箱注册
3. 粘贴你的产品链接（Shopify 产品页链接）
4. Quickfire 自动拉取产品信息
5. 输入提示词（或使用 Claude 生成的广告脚本）
6. 选择格式：TikTok 竖屏 / 横屏 / 35s
7. 等待 AI 生成视频（会自动合成 3 个片段）
8. 预览 → 下载（免费版有水印，付费可去水印）

**广告脚本示例**（用 Claude 生成）：

```
[开头 0-3s] 画面：痛点切入 - 制造停下来的理由
[主体 3-15s] 产生共鸣 - 打破认知
[结尾 15-30s] 给出解决方案 - 产品展示 + CTA
```

## 第六阶段：订单处理与发货

Zendrop 一件代发，自动处理发货。

### 方式一：AI 自动处理（推荐）

在 Claude 中直接发送：

```
请帮我处理今天所有的待发货订单
```

Claude 通过 MCP：

1. 拉取所有待发货订单
2. 确认数量和费用
3. 自动执行发货

### 方式二：手动处理

1. Zendrop 后台 → 左边「订单」
2. 所有 Shopify 订单自动同步显示
3. 点击右上角「发货」
4. Zendrop 自动完成代发

## 附录：提示词模板

### 提示词1：选品研究提示词

```
你是一名顶级的电商产品研究专家，专门研究那些通过 Facebook 广告快速放量的产品。
请使用 Zendrop 的 MCP 连接，在以下细分赛道进行产品的研究：

细分赛道：[填写你的品类，如：电子产品/宠物用品/运动户外]

要求：
1. 搜索多个关键词，覆盖不同的子类目
2. 对每个产品从以下维度打分（1-10）：
   - 情绪购买出发点
   - 解决问题能力
   - 是否适合做 UGC
   - 目标受众清晰度
   - 潜在毛利空间
3. 输出格式：
   - 产品名称
   - 建议零售价
   - Zendrop 成本价
   - 毛利百分比
   - 情绪购买动机分析
   - 目标受众画像
   - 店铺模式建议
   - UGC 视频拍摄角度建议
   - 建议日预算
4. 最终推荐 TOP 3 产品，标注优先测试顺序
```

### 提示词2：深度市场研究提示词

```
请针对以下产品进行深度的竞品分析、评论分析、社会舆论分析、广告心理分析、消费者行为分析：

产品名称：[填写产品名称]
产品链接：[填写 Zendrop 产品详情页链接]

分析框架：

1. 市场情报
   - 当前 Facebook 广告主有哪些
   - 竞品主流广告角度（价格对比/开箱痛感测试/技术参数堆砌等）

2. 消费者心理洞察
   - 核心购买动机（情绪驱动）
   - 核心购买顾虑
   - 消费者身份认同分析

3. 差异化营销角度
   - 给 5 个独特营销角度（与互联网上已经存在的内容不同）
   - 每个角度打评分
   - 标注每个角度对应的目标顾客画像
   - 建议优先测试哪个方案

4. 目标受众详细画像
   - 年龄/性别/收入/兴趣爱好
   - 消费场景
   - 情绪触发点
```
